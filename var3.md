\# Вариант 3 — поля ввода в визуализаторе



\## Вкладка «Волновое»



| Поле | Вводить |

|---|---|

| Длина L | `3` |

| Скорость a | `1` |

| Членов ряда N | `20` |

| Начальное условие f(x) | `1 - x/3 + sin(pi\*x/3)` |



\---



\## Неоднородные условия



| Поле | Вводить |

|---|---|

| α(t) = u(0, t) | `1` |

| β(t) = u(L, t) | `(t/10)^2 - t/10` |



\---



\## Правая часть



| Поле | Вводить |

|---|---|

| f(x) — правая часть | `x - 2` |



\---



\## Что выключить / включить



\- \[x] α(t), β(t) — ненулевые концы → \*\*включить\*\*

\- \[ ] ψ(x) — ненулевая начальная скорость → \*\*выключить\*\*

\- \[x] f(x) — правая часть → \*\*включить\*\*



\---



\## Контрольные значения для проверки



| Момент времени | Что проверить |

|---|---|

| t = 0 | max ≈ 1.5511, левый конец = 1, правый = 0 |

| t = 5 | правый конец ≈ −0.25 |

| t = 10 | правый конец = 0 (вернулся) |

| t = 20 | правый конец = 1 |
